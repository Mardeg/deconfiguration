Motorola RFS-6010-1000-WR

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, set baud rate (Speed) to 19200, leave other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Start the Command Line Interface (CLI) mode.

    After pressing Enter when prompted, type cli at the login:  prompt.

  3. Erase configuration and reboot device.

    At the Username:  prompt type restore then for the password type restoreDefaultPassword which is case sensitive:
    User Access Verification Username: restore Password: restoreDefaultPassword
    After a warning message, type y when prompted to continue:
    WARNING:This will wipe out the configuration (except license key) and user data under "flash:/" and reboot the device Do you want to continue? (y/n):y Switch rebooting. Will come up with default configuration. The system is going down NOW !! % Connection is closed by administrator! Please stand by while rebooting the system.
    More output will take you again to the login:  prompt.

  4. Check default login credentials work and list any licenses.

    Repeat Step 2:
    Please press Enter to activate this console. nma-acton-m125-wc01 login: cli Welcome to CLI
    At the Username:  prompt type admin then for the password type superuser all lowercase:
    User Access Verification Username: admin Password: superuser
    At the RFS6000> prompt type the show licenses command:
    RFS6000>show licenses Serial Number 10053520400314 feature license string license value usage AP 2a8633d031d79c477a564811a4e7ed090f98 8 0 b5464b59d7ae046f1eb89fa70dd49f54a1a1 111d1286
    Note: The default IP address for the device via web-interface and SSH is 10.1.1.100 but the web-interface being java-based is obsolete, so SSH access from PuTTY is identical to console access:
    Login as: cli Username: admin Password: superuser Welcome to CLI RFS6000>

  5. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  6. End the session.

    Close PuTTy.
