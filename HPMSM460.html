HP MSM460

  1. Power up the device and do a hard reset.

    Plug in a Power over Ethernet cable to the PoE port of the Access Point from the "To AP" port of a power injector device.
    Allow time for the device to boot, then insert a straightened paperclip under the ethernet cable into hole for the recessed reset button:
    Hold the reset button down and watch the indicator lights blink slowly in unison three times.
    Release the paperclip when they are still lit on the third blink:

  2. Log in with the default credentials.

    Plug in another ethernet cable into an ethernet port of a PC from the "To Switch" port of the power injector.
    Manually set the ethernet PC interface IP to the same subnet as the default IP of the Access Point which is 192.168.1.1
    Open PuTTY and select "SSH" connection type, and enter 192.168.1.1 into the "Host Name (or IP address)" box, then click the "Open" button.
    Type admin at both the login as:  and password:  prompt:
    login as: admin Keyboard-interactive authentication prompts from server: | Password: admin End of keyboard-interactive prompts from server Authorized access only. This system is property of [COMPANY NAME]. Contact [EMAIL] for more information.

  3. Ensure Access Point can operate stand-alone.

    If the End User License Agreement is not shown, then the device is running in controlled mode and needs to be changed to autonomous mode. Type enable at the CLI>  prompt for elevated privileges, then type switch operational mode:
    CLI> enable CLI# switch operational mode
    This reboots the device, so log back in with the default credentials again to check that the EULA appears:
    login as: admin Keyboard-interactive authentication prompts from server: | Password: admin End of keyboard-interactive prompts from server Authorized access only. This system is property of [COMPANY NAME]. Contact [EMAIL] for more information. MSM430 V. 6.3.0.2-19149 (C) 2013 Hewlett-Packard Development Company, L.P. End User License Agreement ========================== PLEASE READ CAREFULLY BEFORE USING THIS EQUIPMENT:
    Further output should bring you to the prompt to accept the EULA:
    To the extent the terms of any HP policies or programs for support services conflict with the terms of this EULA, the terms of this EULA shall control. Do you accept (YES/NO)?>
    Do not continue, just turn off/unplug the device.

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Close PuTTy.
