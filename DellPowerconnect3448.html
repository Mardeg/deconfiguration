Dell POWERCONNECT 3448

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Access Startup Menu

    Press Esc before the countdown ends:
    Autoboot in 2 seconds - press RETURN or Esc. to abort and enter prom.

  3. Bypass current configuration and boot.

    When the Startup Menu appears, type 3 to ignore the current password, then press Enter to boot:
    Startup Menu [1] Download Software [2] Erase Flash File [3] Password Recovery Procedure [4] Enter Diagnostic Mode [5] Set Terminal Baud-Rate [6] Back Enter your choice or press 'ESC' to exit: 3 Current password will be ignored! ==== Press Enter To Continue ==== Exit from boot menu. Continue with flow. Preparing to decompress... 100% Decompressing SW from image-1 1024000 OK Running from RAM... ********************************************************************** *** Running SW Ver. 1.0.0.112 Date 20-Jun-2005 Time 22:04:47 *** **********************************************************************
    Further output will take you to the Dell Easy Setup Wizard.

  4. Erase configuration and reboot.

    When prompted to enter the setup wizard type n
    Would you like to enter the setup wizard (you must answer this question within 60 seconds)? (Y/N)[Y] N
    At the console>  prompt type enable to access privileged mode:
    console> enable
    At the console#  prompt type del startup-config and confirm with y:
    console# del startup-config Delete startup-config [y/n]? y 01-Jan-2000 01:01:42 %FILE-I-DELETE: File Delete - file URL flash://startup- config Startup file was deleted
    Type show vlan to see if any non-default vlans exist:
    console# show vlan Vlan Name Ports Type Authorization ---- --------------- --------------------------- ------------ ------------- 1 1 ch(1-8) other Required 2 name2 g(4-10,12-13,16-17,19,22, permanent Required 24) 100 name100 g(1-9,24) permanent Required 300 name300 g(15,24) permanent Required 600 name600 g(4-9,14,18,20-21,23-24) permanent Required 602 name602 g(11,24) permanent Required 604 name604 g(4-9,24) permanent Required 666 name666 g(4-9,24) permanent Required
    Delete non-default vlans:
    console# configure console(config)# vlan database console(config-vlan)# no vlan 2 console(config-vlan)# no vlan 100 console(config-vlan)# no vlan 300 console(config-vlan)# no vlan 600 console(config-vlan)# no vlan 602 console(config-vlan)# no vlan 604 console(config-vlan)# no vlan 666 console(config-vlan)# exit console(config)# exit
    Type the reload command and confirm with y until it does the system reset:
    Console# reload You haven't saved your changes. Are you sure you want to continue ? (Y/N)[N] Y This command will reset the whole system and disconnect your current session. Do you want to continue ? (Y/N)[N] Y ************************************************** ***************** SYSTEM RESET ***************** **************************************************

  5. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  6. End the session.

    Close PuTTy.
