Dell COMPELLENT SC220

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.
    Ensure through the BIOS settings (accessed with a USB keyboard and monitor via a VGA cable connected to the device) that console redirection is enabled.

  2. Access the System Setup menu.

    Press F2 as the POST screen appears before bootup starts:
    Phoenix ROM BIOS PLUS Version 1.10 2.2.2 Dell Inc Compellent SC8000 www.Dell.com Two 2.50 GHz Six-core Processors, Bus Speed:7.20 GT/s, L2/L3 Cache:1.5 MB/15 MB System running at 2.50 GHz System Memory Size: 64.0 GB, System Memory Speed: 1333 MHz, Voltage: 1.35V Dell Serial ATA AHCI BIOS Version 1.0.2 Copyright (c) 1988-2014 Dell Inc. No supported device found AHCI BIOS not installed QLE2562 PCI3.0 Fibre Channel ROM BIOS Version 3.00 Copyright (C) QLogic Corporation 1993-2010. All rights reserved. www.qlogic.com

  3. Erase configuration.

    Use down/up arrows to enter the iDRAC Settings menu, then select Reset iDRAC configurations to defaults:
    Dell Inc | System Setup F1 for Help iDRAC Settings iDRAC Settings > Reset iDRAC configurations to defaults Resetting to defaults is in progress. 47 % Service Tag: 6SCYWX1 Compellent SC8000
    Select CONTINUE and press Enter when the progress of resetting to defaults reaches 100%
    CONTINUE Reset configurations to defaults completed successfully.

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.
  5. End the session.
    Close PuTTy.
