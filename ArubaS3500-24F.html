Aruba S3500-24F

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Access the configuration commandline.

    When prompted to login, enter password as the user and forgetme! as the password:
    Initializing TPM and Certificates TPM and Certificate Initialization successful. ******************************************** Starting Console Session Retrieving Configuration ... ******************************************** (ArubaS3500-24F) User: password Password: forgetme!
    At the (ArubaS3500-24F) > prompt enter enable as both the command and the password:
    (ArubaS3500-24F) >enable Password:enable
    At the (ArubaS3500-24F) # prompt type the configure terminal command:
    (ArubaS3500-24F) #configure terminal Enter Configuration commands, one per line. End with CNTL/Z

  3. Erase the configuration.

    At the (ArubaS3500-24F) (config) # prompt type the mgmt-user admin root command then enter a temporary password:
    (ArubaS3500-24F) (config) #mgmt-user admin root Password:r00tPassword Re-Type password:r00tPassword
    Exit to the login prompt then enter admin for the user and enter the temporary password you just set:
    (ArubaS3500-24F) (config) #exit (ArubaS3500-24F) #exit (ArubaS3500-24F) >exit (ArubaS3500-24F) User: admin Password: r00tPassword
    At the (ArubaS3500-24F) > prompt enter enable as both the command and the password:
    (ArubaS3500-24F) >enable Password:enable
    At the (ArubaS3500-24F) # prompt type the write erase command then y to confirm:
    (ArubaS3500-24F) #write erase All the configuration will be deleted. Press 'y' to proceed :y Write Erase successful
    Type the reload command then y to confirm:
    (ArubaS3500-24F) #reload Do you really want to restart the system(y/n)? y System will now restart! (ArubaS3500-24F) # Shutdown processing started Syncing data....done. Sending SIGKILL to all processes. Please stand by while rebooting the system. 0:<7>sd 0:0:0:0: shutdown 0:<7>ppe 0000:01:00.0: shutdown 0:<0>Restarting system. 0:. 0:<2>Performing hard reset...
    Upon reboot the system asks for the default user/password which is admin/admin123

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Close PuTTy.
