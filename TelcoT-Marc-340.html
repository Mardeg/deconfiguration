Telco T-Marc-340

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Access the command line interface.

    Press any key when prompted to stop auto-boot and before the countdown ends:
    BATM Telco Boot Loader Switch model : T-Marc-340 Loader version : 8.2.0 created Nov 18 2007 - 14:53:35 MAC Address : 00:A0:12:D2:9D:F0 Press any key to stop auto-boot... 4 start CLI
    Type batm for the password to get to the Loader> prompt:
    User Access Verification Password:batm Loader>

  3. Erase logs/startup configuration and reboot.

    Type the config command to be able to enter the clean log-history and clean startup-config all followed by exit commands, then type the reload command:
    Loader>config Loader(config)#clean log-history Loader(config)#clean startup-config all Loader(config)#exit Loader>reload Wait for restart...
    After the banner is shown press the Esc key to stop the Zero Touch process when prompted and before the countdown ends:
    auto-booting... Uncompressing 4084057 bytes... Loading image... 14520864 BUILT-IN SELF TEST ------------------ CPU Core Test : Passed Power Supply Test : Passed Fan Test : Passed /////////////////////////////////////////////////////////////////////////// // // // // // B A T M A d v a n c e d C o m m u n i c a t i o n s // // // // T e l c o S y s t e m s // // // // Device model : T-Marc-340 // // Product Category : AccessEthernet(TM) // // SW version : 10.5.R1 created May 29 2013 - 12:08:52 // // // // // /////////////////////////////////////////////////////////////////////////// Changing interface SW0 IP address to 10.0.252.191/16 ############### Press ESC to stop Zero Touch process... 8

  4. Log in with default credentials and perform a factory reset.

    Type batm at the Password: prompt:
    User Access Verification Password:batm
    Type the enable command to get to the T-Marc_3X0# prompt:
    T-Marc_3X0>enable T-Marc_3X0#
    Type the reload to-defaults command and confirm with the y key:
    T-Marc_3X0#reload to-defaults Restore factory setting and reboot the Switch ? [y/n] : y Rebooting ...

  5. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  6. End the session.

    Close PuTTy.
