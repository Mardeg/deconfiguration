Netgear ProSAFE WNDAP360

  1. Start the console session.

    Plug in serial cable to device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Power up the device and do a hard reset.

    Using paperclip wire hold down reset button on side of device:
    Release the button after 5+ seconds.

  3. Erase the configuration and reboot.

    Allow the device to boot up and when prompted type admin for the login and password for the password:
    System initilization is .. [DONE...] Starting SysMon Script... [DONE] Welcome to SDK. Have a lot of fun... netgearAB6928 login: admin Password: password
    Type the restore-factory-default command:
    netgearAB6928#restore-factory-default root: Restoring configuration from file /etc/default-config. root: Generating AP name. root: CLOUD: Making a fresh copy of pal.cfg Sending SIGKILL to all processes Requesting system reboot Restarting system.

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Turn off/unplug the device and close PuTTy.
