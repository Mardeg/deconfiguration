Dell POWEREDGE R410

  1. Start the console session.

    Plug a monitor into device using a VGA cable.
    Plug a USB keyboard and mouse into device USB ports.
    Plug power into device and if a power button exists, turn it on.

  2. Reset iDRAC to factory defaults

    Press F10 for System Services when the option appears. Notice that there is an iDRAC IPv4 address set to something other than 192.168.0.120 indicating the settings are non-default. When prompted press Ctrl E for iDRAC6 Configuration Utility:

  3. Erase System Settings and reboot.

    Navigate to Hardware Configuration > Delete Configuration and Reset Defaults:
    Use spacebar to toggle selection, then either Tab to the Finish button and press Enter or click on it and confirm in the dialog with the Yes button:
    The system may restart or shut down when the progress bar completes:

  4. End the session.

    Unplug all cables.
