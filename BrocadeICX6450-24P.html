Brocade ICX 6450-24P

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Access boot monitor prompt.

    Repeatedly press the b key before the countdown ends. Example output:
    Bootloader Version: 10.1.01T310 (Apr 9 2014 - 18:01:02) Model ID: 0.0.0.1.0.0 Enter 'b' to stop at boot monitor: 0 ICX64XX-boot>>

  3. Bypass any configured login.

    Type the no password command:
    ICX64XX-boot>> no password OK! Skip password check when the system is up.
    Type the boot command:
    ICX64XX-boot>> boot Booting image from Primary ## Booting image at 00007fc0 ...
    The boot process should list some system details including licenses, then bring you once again to a command prompt.
    ========================================================================== UNIT 1: SL 1: ICX6450-24p POE 24-port Management Module Serial #: BZR0440J07T License: ICX6450_BASE_ROUTER_SOFT_PACKAGE (LID: dbtFJJFlFMv) P-ENGINE 0: type DEF0, rev 01 ========================================================================== UNIT 1: SL 2: ICX6450-SFP-Plus 4port 40G Module ========================================================================== 800 MHz ARM processor ARMv5TE, 400 MHz bus 65536 KB flash memory 512 MB DRAM A1HRCSW1>

  4. Erase all configuration settings.

    Type the enable command for elevated privileges, then show license to list any features/keys installed:
    A1HRCSW1>enable No password has been assigned yet... A1HRCSW1#show license License record empty
    Type the erase startup-config command:
    A1HRCSW1#erase startup-config: Erase startup-config Done. dhcp server lease database is also removed

  5. Reboot and check there is no login prompt.

    Type the reload command, and confirm with y whenever prompted:
    A1HRCSW1#reload Are you sure? (enter 'y' or 'n'): y Could not verify if the Running Config data has been changed. Do you want to continue the reload anyway? (enter 'y' or 'n'): y Halt and reboot Sent SIGKILL to all processes Requesting system reboot Restarting system.
    Further output should bring you to the sampICX6430-48 Switch> prompt.
    Type the enable command, then the show users command:
    ICX6430-48 Switch>enable No password has been assigned yet... ICX6430-48 Switch#show users No username entries

  6. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  7. End the session.

    Turn off/unplug the device and close PuTTy.
