EMC² DS-32 M2

  1. Start the console session.

    Plug in ethernet cable to ethernet port of device from PC ethernet port
    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, change baud rate (Speed) to 57600 and leave other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Reset password

    Press Enter when you see the line SNMP: trap_task begins... to bring up the > prompt, then type password to continue:
    SNMP: trap_task begins... Fabric Paths Available Fabric Processes Completed >password SP0: Subsystem initialization starting
    At the C> prompt type the pswd_reset command:
    C>pswd_reset SSP0: Loading UUA.EL1......... DONE SSP0: Loading SSP.L21......... DONE
    The device will reboot and this time should bring you directly to the C> prompt.

  3. Log in from the Web interface.

    Set the IP, subnet and gateway of the device:
    C>ipconfig 192.168.1.1 255.255.255.0 192.168.1.0 Request initiated. C> Request Completed: OK
    Manually set the ethernet PC interface IP to the same subnet as the IP you just set the device to:
    In your web browser addressbar type http://192.168.1.1
    When prompted enter the case sensitive default username: Administrator and password: password to get the Web interface.
  4. Erase the configuration.
    Click the Continue button at the "use Netscape Navigator" warning, then click the Details button.
    Navigate in the menu: Maintenance > Switch, then click the Deactivate button for "Online state"
    In the "Erase" section click the Reset Configuration button and click the OK button in "Are you sure that you want to reset the configuration?" dialog.
    You should see the following output in PuTTY and the device will reboot:
    C>LGN0: 08/17/22 09:35:56: All ports logged out of Fabric, switch going offline. ************ Reseting EWS NVRAM ************ Reseting CLI NVRAM SSP0: Loading UUA.EL1......... DONE SSP0: Loading SSP.L21......... DONE SSP0: Subsystem initialization starting code end=E0956D78, lower mem top = E0E00000

  5. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  6. End the session.

    Close PuTTy.
