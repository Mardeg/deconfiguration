NEC SN8029 2-port Analog Media Converter

  1. Start the console session.

    Plug in ethernet cable with RS232 DB-9pin adaptors to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Access the Maintenance command line.

    Press the Enter key when initial boot sequence ends, then type the usrConsole command:
    Boot program version: SP3651_ver9.00 MC-ID: f8b7972d840e Starting 2A-MC boot program....... --- Online start --- (wait 30sec) Attached TCP/IP interface to axDev unit 0 Attaching network interface lo0... done. NFS client support not included. Adding 4770 symbols for standalone. sysdata is the latest version! This H/W is Revision 2 [createRamDrv]New File System is Created ------//----------------------------------------//--- 2 Port Analog Media Converter *Firmware SP-3753 ( 25.00) *Compiled on Dec 11 2014 18:33:52 ---//----------------------------------------//------ -> 0x8d59754 (tNetTask): 0x8d59754 (tNetTask): duplicate IP address 00000000 sent from ethernet address 40:a8:f0:62:89:a5 duplicate IP address 00000000 sent from ethernet address 40:a8:f0:62:89:a5 -> usrConsole 2A-MC Maintenance command Version 1.00 2001.07.19

  3. Erase the configuration data.

    Type the init config command, confirming with y twice:
    2A-MC > init config *** Warning !! *** This command is for initialize config data. If this command is excuted, your configuration is deleted!! OK ?[Y/N] : y It's recommended to back up your configuration !! really OK ?[Y/N] : y Initialize of configuration data was successful. You need to setup MC2's config data!!
  4. Copy all the output from the session into your records.
    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Close PuTTy.
