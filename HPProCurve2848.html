HP ProCurve 2848 J4904A

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Use hardware buttons to initiate a factory reset.

    Allow the system to boot up, pressing Enter twice when prompted to get to the Username:  prompt. If there is no log in prompt skip to step 3:
    ROM information: Build directory: /sw/rom/build/makrom(swt_ex) Build date: Aug 3 2005 Build time: 11:15:19 Build version: I.08.07 Build number: 8210 OS identifier found at @ 0xff800000 Verifying Image validity ... CRC on OS image header Passed CRC on complete OS image file Passed Valid OS image @ 0xff800000 Decompressing...done. initializing......initialization done. Waiting for Speed Sense. Press <Enter> twice to continue. Connected at 9600 baud Username:
    Using paperclip wires, press both the Reset and Clear buttons down together until you see LEDs light up as a response, release the Reset button.
    When “Test” blinks fast release the Clear button. You'll know if it worked when the system reboots and pressing Enter twice no longer prompts for a username, instead you are presented with a menu:
    Waiting for Speed Sense. Press <Enter> twice to continue. Connected at 9600 baud ProCurve Switch 2848 1-Jan-1990 0:01:58 ==========================- CONSOLE - MANAGER MODE -============================ Main Menu 1. Status and Counters... 2. Switch Configuration... 3. Console Passwords... 4. Event Log 5. Command Line (CLI) 6. Reboot Switch 7. Download OS 8. Run Setup 9. Stacking... 0. Logout To select menu item, press item number, or highlight item and press <Enter>.

  3. Ensure data has been erased.

    Type 5 for the Command Line and at the ProCurve Switch 2848#  prompt type the show vlans command followed by the erase startup-config command and confirm with y:
    ProCurve Switch 2848# show vlans Status and Counters - VLAN Information Maximum VLANs to support : 8 Primary VLAN : DEFAULT_VLAN Management VLAN : 802.1Q VLAN ID Name Status Voice Jumbo -------------- ------------ ---------- ----- ----- 1 DEFAULT_VLAN No No ProCurve Switch 2848# erase startup-config Configuration will be deleted and device rebooted, continue [y/n]? y Rebooting the System

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Close PuTTy.
