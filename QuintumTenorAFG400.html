Quintum Tenor AFG400

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Erase configuration and log in with default credentials.

    Press r for Factory Defaults when prompted and before the countdown ends.

All Rights Reserved. | | | | Hardware ID: 0111 (7) | | | +--------------------------------------------------------------------------+ Loading software: Length = 3961538, Checksum = 0xb7c2. Please wait... Port A MAC Address: 00:30:E1:20:82:70 Attaching network interface lo0... done. VxWorks Copyright 1984-2001 Wind River Systems, Inc.     Press y to confirm when prompted and before the countdown ends.
    PRESS 'y' WITHIN 4 SECONDS TO CONFIRM. FACTORY DEFAULTS RESTORED. Initializing Database... Loading default Database ........................................... Loading Help... Loading Messages... Done. The system is starting up ....
    Type admin for both the Login and Password prompts:
    <a3defa06> Login:admin Password:admin
    Type the config then setfactory commands and confirm with yes:
    Quintum# config config# setfactory Are you sure that you want to set configuration to factory default (Yes/No)?yes Loading default Database ............................................. loaded System configuration is set to factory default. Please reboot the system now. config#

  3. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  4. End the session.

    Close PuTTy.
