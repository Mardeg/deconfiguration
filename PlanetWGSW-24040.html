Planet WGSW-24040

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.
  2. Perform a hardware reset.
    Using a paperclip wire press the recessed reset button and hold it down for 10+ seconds.

  3. Log in with default credentials.

    When prompted type admin for both Username and Password:
    + SGSW/WGSW-24040 Bootloader v1.0 - built Aug 19 2008 RAM: 0x00000000 - 0x04000000, FLASH: 0x80000000 - 0x80800000, 256 blocks of 0x00010000 bytes each. memtest Testing [0x0002c340-0x03fe1000]... Completed (no errors) fis load managed go Welcome to PLANET Command Line Interface.   4. Erase configuration

    Type system to access the System commands:
    SWITCH/>system Type 'up' or 'exit' to move up one level or '/' to go to root level
    Type the restore default command:
    SWITCH/System>restore default

  5. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  6. End the session.

    Close PuTTy.
