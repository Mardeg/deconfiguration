AudioCodes Mediant 800b

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Reset passwords.

    Allow the device to boot up to the login prompt:
    ## Loading Linux kernel with entry point: 0x8056e000 ... Bootloader: Done loading app on coremask: 0x3 ......... Loading Application ....................................... Invoking Startup ........................................................................................ Welcome to AudioCodes CLI Username:
    Use a paperclip to press and hold down the "Reset" pinhole button for 12+ seconds:
    Allow the device to boot back up to the login prompt again.

  3. Login with default credentials and erase configuration.

    Type Admin for both Username and Password, this will bring you to the M800B>  prompt
    ## Loading Linux kernel with entry point: 0x8056e000 ... Bootloader: Done loading app on coremask: 0x3 ......... Loading Application ....................................... Invoking Startup ........................... Welcome to AudioCodes CLI Username: Admin Password: Admin M800B>
    Type the enable command for elevated privileges, and Admin when prompted for a password to get to the M800B#  prompt:
    M800B> enable Password: Admin M800B#
    Type the show system feature-key command to check what licensed features are active:
    M800B# show system feature-key Key features: Board Type: M800B Channel Type: RTP DspCh=150 HA Coders: G723 G729 GSM-FR G727 G722 PSTN Protocols: ISDN IUA=2 CAS DATA features: DSP Voice features: IpmDetector Security: IPSEC MediaEncryption StrongEncryption EncryptControlProtocol IP Media: VXML Control Protocols: MSFT FEU=800 SIP SBC=65 Default features: Coders: G711 G726
    Type the write factory command to erase the configuration and restart:
    M800B# write factory Writing factory default and restarting... M800B# **AUDC*** end of serial init U-Boot 1.1.1 (Development build) (Build time: Mar 25 2015 - 12:41:17) AudioCodes uKernel U-Boot Version: MP500 K9

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Turn off/unplug the device and close PuTTy.
