APC NETBOTZ RACK MONITOR 200

  1. Start the console session.

    Plug in serial cable to console port of device from PC COM1 port
    Open PuTTY and select "Serial" connection type, leave baud rate (Speed) and other settings as default.
    Plug power into device and if a power button exists, turn it on.

  2. Log in and erase configuration data.

    Type apc for both the User Name and password prompts:
    User Name : apc Password : apc American Power Conversion Network Management Card AOS v3.5.9 (c) Copyright 2008 All Rights Reserved NetBotz 200 App v3.5.6 ------------------------------------------------------------------------------- Name : Unknown Date : 06/16/2011 Contact : Unknown Time : 12:08:09 Location : Unknown User : Administrator Up Time : 0 Days 0 Hours 5 Minutes Stat : P+ N? A+ -------------------------------------------------------------------------------
    In the Control Console menu type 3 for the System menu:
    ------- Control Console ------------------------------------------------------- 1- Device Manager 2- Network 3- System 4- Logout <ESC>- Main Menu, <ENTER>- Refresh, <CTRL-L>- Event Log > 3
    In the System menu type 4 for the Tools menu:
    ------- System ---------------------------------------------------------------- 1- User Manager 2- Identification 3- Date/Time 4- Tools 5- RADIUS 6- Modem 7- About System <ESC>- Back, <ENTER>- Refresh, <CTRL-L>- Event Log > 4
    In the Tools menu type 2 to Reset to Defaults and confirm with yes:
    ------- Tools ----------------------------------------------------------------- 1- Reboot Management Interface 2- Reset to Defaults 3- Reset to Defaults Except TCP/IP 4- Reset Only TCP/IP to Defaults 5- Delete SSH Host Keys and SSL Certificates 6- File Transfer <ESC>- Back, <ENTER>- Refresh, <CTRL-L>- Event Log > 2 ----------------------------------------------------------------------- Reset to Defaults Enter 'YES' to continue or <ENTER> to cancel : yes Now initializing system to default values including all User Names, Passwords and TCP/IP settings. Please wait... Done.
    Keep pressing the Esc key till you are again at the Control Console menu and type 4 to logout:
    ------- Control Console ------------------------------------------------------- 1- Device Manager 2- Network 3- System 4- Logout <ESC>- Main Menu, <ENTER>- Refresh, <CTRL-L>- Event Log > 4

  3. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  4. End the session.

    Close PuTTy.
