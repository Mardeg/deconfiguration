Ruckus R510 Access Point

  1. Power up the device and do a hard reset.

    Plug in a Power over Ethernet cable to the PoE port of the Access Point from the "To AP" port of a power injector device:
    Use a straightened paperclip to hold down the "HARD RESET" button on side of device:
    Release the button after 10+ seconds.

  2. Log in with the default credentials.

    Plug in another ethernet cable into an ethernet port of a PC from the "To Switch" port of the power injector.
    Manually set the ethernet PC interface IP to the same subnet as the default IP of the Access Point which is 192.168.0.1
    Open PuTTY and select "SSH" connection type, and enter 192.168.0.1 into the "Host Name (or IP address)" box, then click the "Open" button.
    Type super at both the login as:  and Please login:  prompts, then sp-admin at the password :  prompt:
    login as: super Please login: super Password: sp-admin Welcome to Ruckus Unleashed Network Command Line Interface

  3. Erase configuration.

    Type the enable command:
    ruckus> enable
    Type the set-factory command:
    ruckus# set-factory Are you sure you want to restore Unleashed(and your network) to the default state[Y/n]y ruckus#

  4. Copy all the output from the session into your records.

    Right-click PuTTY's titlebar, from that dropdown menu click on "Copy All to Clipboard"
    Open the file of record certifying the deconfiguration of this device and use CTRL V to paste all the output from PuTTY into that file. Formatting the text with a fixed-width font helps it look more readable.

  5. End the session.

    Unplug the Power Injector and ethernet cables and close PuTTy.
